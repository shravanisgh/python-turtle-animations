1. Go to: https://www.python.org/downloads/  
2. Download Python 3.x for your operating system  
3. During installation, **check the box**:  
   ‚Üí *Add Python to PATH*

After installation, open Command Prompt / Terminal and verify:
```bash

Step 2: Install Visual Studio Code (VS Code)

Download and install VS Code

Open VS Code after installation

Step 3: Open the Project Folder in VS Code
Download or clone this repository from GitHub

heart.py

rbg heart.py

spiral.py

Step 4: Run an Animation
In VS Code, open any file (for example: heart.py)

Open the terminal inside VS Code:

Press: `Ctrl + `` (backtick)

Run the file using:

bash

python heart.py
For the other animations, run:

bash
